公司白話介紹與一句話結論

Belden 是工業網路、資料傳輸、連接線纜、企業網通與自動化基礎設施供應商。投資主軸不是單純線材，而是工業自動化、IT/OT convergence、資料中心與高密度 AI 基礎建設的「連接、機櫃、電力、散熱與網路」整合能力。

一句話結論：BDC 不是便宜到閉眼買的低 P/E 股，但在 2026 forward adjusted EPS 基礎上約 14x，若 RUCKUS 收購順利、毛利率與 EPS 確實上修，115 美元附近仍有中期向 129-153 美元重估的空間。

Forward Adjusted EPS 估值帶

12x / 14x / 16x / 19x 不是官方倍數，也不是所有股票都套用的固定模板，而是本次針對 BDC 的研究判斷估值狀態。錨點來自目前 forward P/E 約 13.97x、BDC 6M/1Y current TTM proxy 中位數約 19.8x-20.0x、peer forward P/E 中位數約 25.75x，以及 BDC 槓桿、margin profile 與 RUCKUS 整合風險。因 BDC 尚未完全升級成高 margin 網通平台，所以 Base fair 不直接用 peer forward P/E 中位數 25.75x，也不把 19x 當基本情境；本報先用 16x forward adjusted EPS 作為 Base fair。這不是說 BDC 應該低於自己過去 TTM P/E 中位數，而是因為 forward adjusted EPS 8.06 明顯高於 TTM GAAP EPS 約 5.95，所以 forward multiple 要打折使用。

口徑差異：TTM P/E proxy 19.8x 是用 TTM GAAP EPS 約 5.95 算，對應價格約 117.8；它是近期中位，不是 overheat。Forward adjusted EPS 19x 是用 FY2026E adjusted EPS 8.06 算，對應價格約 153.1；這才是 rerating 上緣。兩者同樣叫 19x，但分母不同，價格意義完全不同。

更白話地說：Base fair = 8.06 x 16 = 129，可以理解成把 forward EPS 打折後再套接近過去的 19x P/E。若用 19x 當歷史錨點，折扣約 16 / 19 = 84.2%，接近 0.85；若用 19.8x 當 6M/1Y proxy 中位，折扣約 16 / 19.8 = 80.8%。

寫法公式結果意義
折價 forward multiple8.06 x 16.0x129.0不直接給 19x forward，因 RUCKUS、margin、槓桿仍待驗證
對 19x 歷史錨點做 EPS haircut8.06 x 84.2% x 19.0x129.0等於只承認約 84% 的 FY2026E adjusted EPS / forward story
對 19.8x 近期 TTM proxy 中位做 EPS haircut8.06 x 80.8% x 19.8x129.0若用 19.8x 當錨點，折扣更接近 81%

打折原因：8.06 是 FY2026E adjusted EPS，不是已實現的 TTM GAAP EPS；RUCKUS 收購、利息費用、整合成效、margin accretion 與需求能見度還沒完全被財報證明。因此 19x forward 先放在 rerating upper，而不是 Base fair。

風控、品質與下一驗證點

  • 短線防守/風險審核線：105-110 美元。這裡接近 1Y current TTM EPS proxy 的壓力區，跌破代表市場重新壓估值，但不是自動砍倉。
  • 嚴格停損候選：100 美元。若跌破且沒有基本面支撐，或同業沒跌但 BDC 自己弱，forward EPS 可信度被市場打折。
  • 第一停利：129-135 美元。若沒有 EPS 上修，可部分停利。
  • 第二停利：150-153 美元。需要 RUCKUS 順利 close、毛利率改善與 EPS 上修。
  • 品質：TTM OCF 328.76M 高於 net income 約 236.61M，現金轉換不差；但收購後槓桿與利息費用是最大風控點。
  • 下一驗證：2026-07-30 盤前財報、RUCKUS H2 2026 close、121 美元 200DMA 是否站回。
